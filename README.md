## Featured Project

### Custom Calendar Journal

**Repository:** [linxxii-proo/internship-calendar](https://github.com/linxxii-proo/internship-calendar)

A cross-platform daily record application for customizable tracking scenarios such as internship logs, study plans, fitness records, and personal progress records.

**What I built**

- Developed the macOS version with **SwiftUI** and **AppKit**.
- Implemented date-based record management, local JSON persistence, image/document attachments, and backup/restore.
- Added PDF and HTML export so records can be converted into shareable reports.
- Added a Windows single-file HTML version using browser local storage for lightweight access.
- Integrated **DeepSeek API** for Chinese note expansion and polishing, while storing the API key locally in macOS Keychain.
- Wrote core tests for calendar calculation, persistence, backup restoration, export generation, and API request/response formatting.
